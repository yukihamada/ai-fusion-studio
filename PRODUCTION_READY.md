## 🚀 プロダクション環境クイックセットアップ

### 最小システム要件
- **CPU**: 8コア
- **RAM**: 16GB
- **ストレージ**: 100GB
- **Python**: 3.8+
- **GPU**: オプション（NVIDIA推奨）

### プロダクションインストール
```bash
# プロダクション対応リポジトリをクローン
git clone https://github.com/yukihamada/ai-fusion-studio.git
cd ai-fusion-studio

# プロダクション依存関係をインストール
pip install -r requirements.txt

# インストール検証
python -m pytest tests/

# プロダクション WebUI を起動
./start_web.sh
```

### ヘルスチェック
